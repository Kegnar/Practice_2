Задание: Система управления фитнес-центром 
Цель: 
Создать простую систему управления фитнес-центром, которая позволяет управлять клиентами, 
тренерами и абонементами. 
Требования: 
1.  Перечисления (Enum): 
• Создайте перечисление MembershipType, которое будет содержать различные типы 
абонементов (например, Monthly, Yearly, DropIn). 
2.  Интерфейсы (Interface): 
    • Создайте интерфейс IPerson, который содержит методы: 
    • void DisplayInfo() — для отображения информации о человеке. 
    • string GetRole() — для получения роли человека (клиент или тренер). 
3.  Структуры (Struct): 
    • Создайте структуру ContactInfo, которая содержит свойства: 
    • string Email 
    • string PhoneNumber 
    • Переопределите метод ToString() для вывода контактной информации. 
4.  Наследование (Inheritance): 
• Создайте базовый класс Person, который реализует интерфейс IPerson и содержит 
следующие свойства: 
    • string FirstName 
    • string LastName 
    • ContactInfo Contact 
    • Реализуйте методы интерфейса IPerson в классе Person. 
    • Создайте производные классы Client и Trainer, которые наследуют от Person: 
    • Класс Client добавляет свойства: 
    • string ClientID 
    • MembershipType Membership 
    • Класс Trainer добавляет свойства: 
    • string TrainerID 
    • List<string> Specialization.  Реализация: 
    • Создайте класс FitnessCenter, который содержит списки клиентов и тренеров 
    (List<Client> и List<Trainer>). 
    • Реализуйте методы для добавления клиента или тренера в фитнес-центр и для 
    вывода информации о всех клиентах и тренерах. 
6.  Тестирование: 
• В методе Main создайте несколько объектов Client и Trainer, добавьте их в фитнес-центр и выведите информацию о них.
